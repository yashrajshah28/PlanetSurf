
# Planet Surf

## Installation Steps

Install PlanetSurf with npm

```bash
    Steps
    Server folder
    1 - Open new terminal and split
    2 - cd server
    3 - npm i
    4 - npm audit fix --force
    After both installation complete type
    5 - nodemon index.js 
   
```
```bash
    Steps
    Client folder
    1 - cd client
    2 - npm i
    3 - npm audit fix --force
    After both installation complete type
    4 - npm start
```
